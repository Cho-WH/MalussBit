지원 브라우저 안내

다음 환경에서 Web Bluetooth 사용 시 성공률이 높습니다.

  • Windows / macOS: Chrome, Microsoft Edge
  • Chromebook: Chrome (최신 버전)
  • Android: Chrome, 삼성 인터넷

iOS/iPadOS는 Safari 기반 브라우저에서 Web Bluetooth를 지원하지 않습니다. Bluefy 등 전용 앱을 이용해야 합니다.
